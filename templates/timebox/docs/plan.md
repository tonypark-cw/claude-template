# Time Boxing Planner - 기획문서

## 개요

일론 머스크의 5분 블록 타임박싱 기법을 실천할 수 있는 단일 HTML 웹 페이지.
하루를 5분 단위 블록으로 나누고, 작업을 배정/추적/완료하는 플래너.

## 핵심 컨셉 (영상 기반)

- **5분 규칙**: 하루를 5분 단위 블록으로 분할 (18시간 = 216블록)
- **우선순위 배치**: 중요도 순으로 작업 배정
- **파킨슨 법칙 역이용**: 시간을 제한해서 집중도를 높임
- **선택 부담 제거**: 미리 계획된 일정을 따르므로 "뭘 할까?" 고민 제거
- **휴식 필수**: 1~2시간마다 10분 휴식 블록 삽입
- **예비 블록**: 긴급 상황/전화 대응용 여유 시간 확보

## 핵심 기능

### 1. 타임라인 (Timeline)
- 시작 시간 + 총 시간 설정 가능 (기본: 06:00, 16시간)
- 5분 단위 블록 생성 (16시간 = 192블록)
- 현재 시간 하이라이트 + "Go to Now" 스크롤
- 과거 블록 흐리게 표시

### 2. 작업 관리 (Task CRUD)
- 작업명 + 블록 수 + 우선순위(high/medium/low) 입력
- Enter키 또는 Add 버튼으로 추가
- x 버튼으로 삭제 (타임라인에서도 제거)
- 색상 자동 배정 (8색 순환)

### 3. 작업 배정 방식
- **클릭 배정**: 빈 블록 클릭 → 모달에서 작업 선택
- **드래그 앤 드롭**: 작업을 타임라인 블록으로 드래그
- **자동 배정**: "Auto-assign" 버튼 → 우선순위 순서로 빈 블록에 자동 배치
- **휴식 배정**: 모달에서 "Break" 버튼 또는 "Add Breaks" 일괄 추가

### 4. 완료 추적
- 배정된 블록 우측 체크 버튼으로 완료 토글
- 진행률 실시간 표시 (프로그레스 바 + 퍼센트)

### 5. 통계 대시보드
- Total Blocks: 전체 블록 수
- Assigned: 배정된 블록 수
- Completed: 완료된 블록 수
- Progress: 완료율 (%)
- Current Time: 실시간 시계

### 6. 데이터 지속성
- localStorage로 자동 저장 (작업, 타임라인, 완료 상태)
- 새로고침/재방문 시 복원
- "Reset All" 버튼으로 전체 초기화

## 기술 스택

- **프론트엔드**: Vanilla HTML/CSS/JS (의존성 없음, 단일 파일)
- **테스트**: Vitest (유닛) + Playwright (E2E, headless)
- **디자인**: GitHub Dark 테마 기반

### 발견 가능한 버그 주시 포인트

1. **Auto-assign 포인터**: ptr이 작업 간 리셋 안 됨 → 순차 배치 (의도적이나 검증 필요)
2. **Break 간격**: `i += interval + breakLen` (26 stride) → 실제로 2시간10분 간격
3. **confirm 다이얼로그**: Reset All 시 Playwright에서 dialog 핸들링 필요
4. **드래그 타겟**: ondrop은 .time-block에, onclick은 .block-content에 → 타겟 정확성
