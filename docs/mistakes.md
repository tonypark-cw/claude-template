# Mistake Log

Record mistakes and learn from them. Never repeat the same mistake.
Format: date / what happened / why / how fixed / lesson learned

---

## Template

```
### [Date] Mistake Title

**Situation**: What were you trying to do?
**Mistake**: What went wrong?
**Cause**: Why did it happen?
**Fix**: How was it resolved?
**Lesson**: How to prevent this next time?
```

---

## Pre-work Checklist

Before any task, verify:

- [ ] Did you read the file to modify first?
- [ ] Do you understand this file's role in the overall structure?
- [ ] Any similar mistakes in this log?
- [ ] Is the current state recoverable? (git commit status)
